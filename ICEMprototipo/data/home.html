Informações iniciais:

Bem vindo a página de Interface do MeteoPIN.

Aqui apresenta-se informações importantes da estação. Para realizar as configurações necessárias navegue pelas página utilizando o Menu no canto superior esquerdo.

Para mais informações, acesse a página "informações". Lá estão presentes dados do projeto bem como a seção de ajuda para aprender a configurar a estação, além de cuidados que se deve tomar ao instalar.

Caso queira reiniciar para as configurações de fábrica, vá até a seção configurações e desça até a sessão "reiniciar configurações".
